Sobre o FP BOX

FP BOX v0.1 é um sistema modular skeleton criado para servir como base para aplicações de produtividade e organização.

Este é um framework offline-first, completamente em HTML/CSS/JavaScript vanilla, projetado para ser extensível através de módulos independentes.

Características:

  • Arquitetura modular plugin-style
  • Navegação sem reload de página (SPA)
  • Carregamento dinâmico de módulos
  • 100% offline (sem dependências externas)
  • Design responsivo e moderno

Desenvolvido como parte do Sprint Lab – Ecossistema FP
